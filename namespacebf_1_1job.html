If you call 'taskEmplaceData' or 'taskSetData' and need to update the data once more be sure to free the previous contents correctly if the data stored in the buffer is not trivial.

enum bf::job::QueueType : std::uint8_t
strong

The priority that you want a task to run at.

Enumerator
MAIN 

Use this value when you need a certain task to be run specifically by the main thread.

HIGH 

Normally you will want tasks to go into this queue.

NORMAL 

Slightly lower priority than 'QueueType::HIGH'.

BACKGROUND 

Lowest priority, good for asset loading. Tasks in this queue will never run on the main thread.
